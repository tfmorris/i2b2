i2b2 Workbench Perspective

The PFT view is part of the default i2b2 Workbench perspective in version 1.0 of the i2b2 Workbench. This perspective is also accessed via the Window > Open Perspectives > i2b2 Workbench context menu.

This perspective provides a set of views for interacting with the PFT web services. The PFT View is the primary user interface for PFT processing.
