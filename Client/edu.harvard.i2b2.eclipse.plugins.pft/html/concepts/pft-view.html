PFT View

The PFT View appears as part of the i2b2 Workbench Perspective. This view is accessed via the Window > Show View > PFT context menu.

Features

This view provides a UI for interacting with the PFT web services. Reports are dragged into the PFT Report tab and then submitted for processing. Results are displayed in the Results tab.

The view itself consists of four distinct areas:

  • The Report Tab containing the report text to be analyzed. This text may be dragged into the text field from the timeline.
  • The Results Tab containing the concepts found by the PFT web service.
  • The Request XML Tab containing the contents of the request message sent to the PFT web service
  • The Response XML Tab containing the contents of the response message returned by the PFT web service
