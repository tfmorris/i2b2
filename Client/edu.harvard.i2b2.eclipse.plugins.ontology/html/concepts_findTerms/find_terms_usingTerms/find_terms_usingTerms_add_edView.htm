Add to Export Data View

The export data view is designed to compile the requested data and display the information in a graphical representation.

 

Figure 3. 31    Export Data View

 

 

Add Folder To Export Data View

1.       Highlight the folder () by clicking on the name of the folder.

2.       While holding the left mouse button down, drag the folder over to the Export Data View.

3.       Drop the folder into the concept list section of the Select Concepts tab.

4.       The folder will now display in the Export Data View.

 

Add Concept To Export Data View

1.       Highlight the concept () by clicking on the name of the concept.

2.       While holding the left mouse button down, drag the folder over to the Export Data View.

3.       Drop the concept into either the concept list section of the Select Concepts tab.

4.       The concept will now display in the Timeline View.
