Create Table from Selected Rows

This creates a sub-table that contains only the selected rows from the selected parent table.  The sub-table references rows in the parent table, and thus will reflect any changes made to that table.  Row selection is also linked between the tables, so that any selection of rows in one table will be reflected in the other.

 

Note: The create table from selected rows feature can be accessed view the Edit Menu or by clicking on the sub-table icon () located on the file and edit toolbar.

 

1.       Click on the table containing the data you want to use to create a new table.

2.       Using <Ctrl>Left Mouse Button select the rows that contain the data to be saved. (Figure 2.49)

3.       Go to the Edit Menu on the main Control Panel and select Create Table from Selected Rows.

4.       The new table leaf () will be created and appear on the main tree view. (Figure 2.50).

5.       To create a table view for this data, simply click on the Table icon () on the view toolbar or select Table View from the View menu.
