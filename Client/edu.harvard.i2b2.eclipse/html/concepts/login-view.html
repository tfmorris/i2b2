Login View

This view is accessed via the Window > Show View > Login View context menu. The i2b2 workbench uses the common framework provided by Eclipse.

Features

This view provides a UI for interacting with the i2b2 Workbench.

The Title bar lists

  • The workbench name
  • The name of the user that is logged in
  • The status of the login session (green = connected) (red = disconnected). Clicking on the status circle will reveal a log of events including normal events and error conditions.
  • A Help button that connects to the i2b2 wiki.
