Select Patients tab

The patients to be used when rendering tables in the Export Data View are defined in the Select Patients tab.  Users can add patients on an individual basis or by adding a group of pre-defined patients known as a Patient Set or Patient List.  In addition to adding patients, users can remove patients from the list and change the order of the patients.

Patient Set Information

Contains information about the patient set(s) used when creating the list of patients to be used when rendering the tables.

Patient Set Names

 

 

Field Name

Description / Use

Set #

Number assigned each set of patients as it is dragged and dropped into the export data view. It is used as a reference point when viewing the patients in the patient list.

Patient Set Name

The name filed in the database for the patient list/set that is being used.

 

 

Patient Information Section

Displays list of patients and the information that will be used when the tables are created.

Description / Use

Row #

Identifies which row the patient will appear in when the tables are rendered.

Decision

Field is currently not used. The only data that will appear at this time is UnD which means Undefined.

PSet#

Identifies which patient set the patient is part of. If the patient can be found in more than one set then the set # for each patient set will be listed.

Unique number for the patient. Used to identify the patient and their associated records.
