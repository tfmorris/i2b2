Value Constraint

As stated earlier, not all concepts have values.  For those concepts that do can have values there are different types of values that may be associated with it.

 

1.       Flag

Flags are defined in the source system when the value is entered.  The following are some examples that can be found in the i2b2 Demo database.

1.       High

2.       Low

3.       Abnormal

 

2.       Text

The text is defined in the source system when the value is entered.  The following are some examples that can be found in the i2b2 Demo database.

4.       Test Not Performed

5.       Borderline

6.       Positive

 

3.       Numeric

The numeric value is defined in the source system when the result is entered.  The following are some example formats that can be found in the i2b2 Demo database.

7.       200

8.       1.3

 

Unlike flag and text values, numeric values use operators, such as greater than or less than when setting constraints for numerical values. The following is a listing of operators currently in use.

 

Operator

Description/Use

Less Than (<)

Any value less than the number entered.

Less Than or Equal To (<=)

Any value less than or equal to the number entered (inclusive).

Equal To (=)

An exact match to the number entered.

Between

Any value in the range of numbers entered.

Greater Than (>)

Any value greater than the number entered.

Greater Than or Equal To (>=)

Any value greater than or equal to the number entered (inclusive).

 

 

SETTING THE VALUE CONSTRAINT (FLAG)

1.       Double click on the field in the value constraint column.

2.       A new window will open (figure 2.19).

 

Figure 2.19

Value Constraint Window (Flag)

 

 

3.       Click the radial dial next to By high/low flag. 

4.       A new field with a drop down selection box will appear (figure 2.20).

 

Figure 2.20

Value Constraint Flag

 

 

5.       Click on the arrow to change the selection or accept the default.

6.       Click on the OK button.

7.       The constraint value entered will now appear in the value constraint column (figure 2.21)

 

Figure 2.21

Value Constraint Set (Flag)
