Add Previous Query

A previous query can be added to the timeline view by simply dragging the query name from one of the following views.

 

Previous Query View

The previous query view displays the queries that were run by the user. 

 

Figure 8. 8      Previous Query View

 

 

Add Previous Query From Previous Query View

1.       Click on the previous query and drag the item over to the Timeline View.

2.       Drop the previous query into the section that displays the list of concepts (see Figure 8.9).

 

Figure 8. 9      Previous Query Drop

 

 

3.       The concept(s) associated with the previous query will display in the list, the patient list will be added and the name of the query will appear at Query Name (Figure 8.10).

 

 

 

Workplace View

Information in the workplace is related to the most common concepts and queries that an individual uses and in essence becomes their personal workplace.

 

Figure 8. 10    Workplace View

 

 

Add Previous Query From Workplace View

1.       Find the previous query you want to add (either in your folder or a shared folder).

2.       Click on the previous query and drag it over to the Timeline View.

3.       Drop the previous query into the section that displays the list of concepts (see Figure 8.9).

4.       The concept(s) associated with the previous query will display in the list, the patient list will be added and the name of the query will appear at Query Name (see Figure 8.10).
