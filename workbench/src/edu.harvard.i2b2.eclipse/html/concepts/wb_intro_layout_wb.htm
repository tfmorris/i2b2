Layout of the i2b2 Workbench

The desktop of the i2b2 workbench contains several sections. Each section is defined below.

 

Figure 1. 9      Layout of the i2b2 Workbench

 

 

Application Title Bar

The Application Title Bar contains the name of the application (i2b2 Workbench) as well as the following buttons.

 

Minimize ()

The Minimize button will remove the window from view. The window is still active and can be restored by clicking on the application name located on the windows taskbar.

 

Maximize ()

The Maximize button will make the window fill the screen.

 

Restore Down ()

The Restore Down button will resize the window so it no longer fills the screen.  You can further change the size of the window by using your mouse to drag the sides of the window in or out.

 

Close ()

The Close button will exit the window and close the application.

 

Warning: if you exit the window you will need to log back in the i2b2 Workbench, there is no way to “restore” the window.

 

 

Application Menu Bar

The Application Menu Bar contains the menus for the i2b2 Workbench Application.

 

File Menu

The File Menu contains the following selections.

 

§  Preferences – set preferences for how to display help.

§  Exit – closes the i2b2 Workbench.

 

 

Figure 1. 10    File Menu

 

 

Window Menu

The Window Menu contains the following selections.

 

§  Open Perspectives – displays a list of available perspectives.

§  Show View - displays a list of i2b2 views that can be added to the desktop.

 

Figure 1. 11    Window Menu

 

 

Help Menu

The Help Menu contains the following selections.

 

§  Help Contents - opens the i2b2 Workbench help window.

§  About i2b2 Workbench - displays information about the i2b2 Workbench Application.

§  Welcome - opens the Welcome page for the i2b2 Workbench.

 

Figure 1. 12    Help Menu

 

 

i2b2 Desktop Title Bar

The i2b2 Desktop Title bar contains the following information.

 

§  Project Name: the name of the project the user is logged into.

§  User Name: the name of the user logged in.

§  Status Indicator: visual indicator for the status of the i2b2 Workbench. Clicking on it will open the i2b2 log.

§  i2b2 Button: clicking on it will open a web browser to the i2b2 website.

 

Figure 1. 13    i2b2 Desktop Title Bar

 

 

i2b2 Desktop

The desktop area of the i2b2 Workbench contains the i2b2 Views that are open.  Each view can be minimized, maximized, and moved to a different area of the desktop (see section on Navigation).

 

Figure 1. 14    i2b2 Desktop Area
