Using Patient Set

A list of patients can be added to other i2b2 views by simply dragging the name of the patient set from the Workplace View to one of the following views.

 

Add Patient Set to Export Data View

1.       Find the set of patients you want to use (either in your folder or a shared folder).

2.       Click on the Patient Set and drag the item over to the Export Data View.

3.       Drop the patient set into the patient information section.

 

Figure 9. 35    Adding a Patient

 

 

4.       The patients included in the set will be listed along with the demographic information included in the PDO.

Add Patient Set From Workplace View

1.       Find the patient set you want to add (either in your folder or a shared folder).

 

Figure 9. 37    Workplace View

 

 

1.       Click on the patient set and drag it over to the Timeline View.

2.       Drop the patient set into the Patient Set field.

 

Figure 9. 38    Adding a Patient Set

 

 

3.       The number of patients in the set will display.
