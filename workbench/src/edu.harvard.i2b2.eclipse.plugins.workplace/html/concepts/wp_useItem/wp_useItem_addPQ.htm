Using Previous Query

A group template can be added to other i2b2 views by simply dragging the group template name from the Workplace View to one of the following views.

 

Add Previous Query To Query Tool View

1.       Find the previous query you want to add (either in your folder or a shared folder).

 

Figure 9. 27    Workplace View

 

 

2.       Click on the previous query and drag it over to the Query Tool View.

3.       Drop the previous query into the Query Name.

 

Figure 9. 28    Dropping Previous Query

 

 

4.       The concept(s) associated with the previous query will display in the appropriate group(s) and the name of the query will appear at Query Name.

 

Figure 9. 29    Previous Query Added

 

 

Note: When you run the query it will receive a new name.  This is to distinguish the first previous query from the second.

 

 

Add Previous Query From Workplace View

1.       Find the previous query you want to add (either in your folder or a shared folder).

 

Figure 9. 30    Workplace View

 

 

2.       Click on the previous query and drag it over to the Timeline View.

3.       Drop the previous query into the section that displays the list of concepts (see Figure 8.9).

 

Figure 9. 31    Previous Query Drop

 

 

4.       The concept(s) associated with the previous query will display in the list, the patient list will be added and the name of the query will appear at Query Name (see Figure 8.10).

 

Figure 9. 32    Previous Query Added
