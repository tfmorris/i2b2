Extract PFT Results

The PFT Plugin allows a user to take any PFT report and send the report to the PFT service which extracts PFT results from the report and sends them back. The PFT Plugin then displays the results to the user. The following steps explain how to extract PFT results from any report.

  1. Clear Text

    From the Report Tab in the PFT View, click on Clear Text Button to remove any text from the text area.

  2. Copy Report Text into Clipboard

    From any other application containing a text report, select the text and copy it to the clipboard.

  3. Paste Report Text into Text Area

    From the Report Tab in the PFT View, right click inside the text area and choose paste from the menu to paste the contents of the clipboard into the text area.

  4. Choose Report Type

    From the Report Tab in the PFT View, make sure the Report Type Drop-Down Menu shows "BWH PFT Report" which is the report type of the sample report.

  5. Get Pulmonary Data

    From the Report Tab in the PFT View, click on the Get Pulmonary Data Button to send the report to the PFT service to be processed.

  6. View Results

    After clicking on the Get Pulmonary Data Button, the view automatically switches to the Results Tab where the results of the PFT service are displayed. These results may include actual PFT results or any error messages.
